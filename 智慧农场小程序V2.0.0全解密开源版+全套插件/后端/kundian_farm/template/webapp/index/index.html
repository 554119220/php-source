关于我们

智慧农场小程序于2018年5月1日正式上线，是国内首款支持用户线上完成选地，种菜，养鸡，收蛋，监控，派送等一 体化服务的移动客户端，由重庆坤典科技有限公司设计，开发，运营。

种菜介绍

农场提供50多种蔬菜，种子取自全球最好的品种，全部采用有机种植的方式，无农药，无化肥，无激素，无保鲜剂， 从源头上解决食品安全问题。农场为每块菜地，菜园都安装了视频监控，种植过程完全透明化，用户通过微信小程序 即可随时随地的查看蔬菜生长情况。为保证蔬菜的新鲜，农场收菜后极速送达，真正实现了从田间直接到餐桌。

畜牧介绍

智慧农场推出畜牧领养，领养鸡鸭，猪牛羊等。认养鸡苗。喂养14个月左右，领取360枚鸡蛋，按购买组号顺序分12 次派送，每次30枚，送蛋结束后，农场将母鸡宰杀派送到家。

愿景

 致力于让人人吃上健康的食品，解决食品安全问题。
