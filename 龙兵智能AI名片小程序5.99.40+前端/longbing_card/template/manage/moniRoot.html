小程序内部链接
  • 首页
    地址：
    longbing_card/pages/index/index
    参数 [1]：

    to_uid 员工ID 　--必填

    参数 [2]：

    from_id 绑定分销关系的上级用户ID 　--选填　( 如不填写则默认为上述所填写的员工ID )

    参数 [3]：

    currentTabBar 页面类型 ( toCard 名片详情页， toNews 动态首页， toCompany 官网首页 ) 　--选填　（如不填写则默认展示名片列表页面）

    例如：
    longbing_card/pages/index/index?to_uid=1
  • 商城首页
    地址：
    longbing_card/pages/shop/index/index
    参数 [1]：

    uid 员工ID 　--必填

    参数 [2]：

    fid 绑定分销关系的上级用户ID 　--选填　( 如不填写则默认为上述所填写的员工ID )

    例如：
    longbing_card/pages/shop/index/index?uid=1&fid=2
  • 商品详情页面
    地址：
    /longbing_card/pages/shop/detail/detail
    参数 [1]：

    id 商品ID 　--必填

    参数 [2]：

    to_uid 员工ID 　--必填

    参数 [2]：

    from_id 绑定分销关系的上级用户ID 　--选填　( 如不填写则默认为上述所填写的员工ID )

    例如：
    /longbing_card/pages/shop/detail/detail?id=1&to_uid=1&from_id=2
